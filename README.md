# NovaBrew Coffee Taste Profile Quiz

A personality-based coffee matching quiz built for NovaBrew Coffee Co. It helps subscribers discover their coffee style through a trend-forward interactive experience and connects them to a matching brew.

## Coffee Personalities

- Golden Hour: soft, sweet, sun-washed energy with an easy glow
- Plot Twist: unexpected, bold, and always a little unpredictable
- Smooth Talker: polished, classic, and quietly confident taste
- After Hours: moody, cinematic, and strongest after dark

## Built With

- Next.js
- Tailwind CSS
- TypeScript

## About

Built as a customer-personalization prototype to help NovaBrew improve subscriber retention through taste-profile matching, stronger identity, and a more memorable subscription experience.

## Project Structure

- `REQUIREMENTS.md`: product brief and quiz direction
